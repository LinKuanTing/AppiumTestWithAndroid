ปริมาณแคลอรี่ที่ควรได้รับในแต่ละวัน：___caloriePart___ แคลอรี่
ธัญพืชไม่ขัดสี พืชหัว ___category1___ ถ้วย
1ถ้วย = 1ถ้วยข้าว (200g) = ข้าวต้ม2ถ้วย
  = บะหมี่/เส้นหมี่2ถ้วย = หมั่นโถขนาดกลาง1ลูก
  = ขนมปัง 2แผ่น = ขนมปัง4ชิ้นเล็ก
  = แผ่นเกี๊ยว12แผ่น = แป้งธัญพืช80g
  = มัน1ถ้วย (มันฝรั่ง เผือก)

ประเภทถั่ว ปลา เนื้อสัตว์ ไข่ ___category2___ ส่วน
1ส่วน = เนื้อสัตว์ 37.5กรัม (ไก่ ปลา หมู เนื้อวัว)(35g)
  = เต้าหู้1ชิ้น (100g) = น้ำเต้าหู้1ถ้วย (260cc)
  = ไข่1ฟอง   = ถั่วเหลือง20g
ประเภทนมไขมันต่ำ ___category3___ ถ้วย
1ถ้วย = นมพร่องมันเนย 1แก้ว (240cc)
  = นมผงพร่องมันเนย3ช้อน (25g)

ประเภทผัก ___category4___ ถ้วย
1ส่วน = ดิบ100g
  = สุกครึ่งถ้วย (ความจุถ้วย240cc)
ประเภทผลไม้  ___category5___ ส่วน
1ส่วน = ส้ม1ลูก (150g) = แอ๊ปเปิ้ล1ลูก (130g)
  = กล้วย1/2ลูก (70g) = ฝรั่ง1/2ลูก (160g)
  = แตงโม1ชิ้น (250g) = ชมพู่2ลูก (170g)
  = มะเขือเทศ23ลูก (175g)
  = องุ่น13ลูก (105g)
  = กีวี่1.5ลูก (115g)
ประเภทไขมันและถั่วธัญพืช ___category6___ ช้อนชา
1ช้อนชา = น้ำมัน1ช้อนชา (5g)
  = ธัญพืช7~8g
